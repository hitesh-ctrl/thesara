Privacy Policy

📅 Last updated: June 2026 ✅ Effective immediately 🔓 Open source

Plain-English Summary

Thesara sends subtitle text to AI and dictionary services only when you hover a word. We do not store anything on our servers. There are no accounts, no analytics, no advertising, and no monetisation of any kind. Your data is not our product.

1. Data collected during normal use

Thesara collects no data passively. The following information is transmitted exclusively at the moment you hover a word or phrase in a Netflix subtitle:

Data Sent to Purpose
Hovered word or phrase Free Dictionary API, Groq Retrieve definition, synonyms, and contextual explanation
Current subtitle line Groq (via Cloudflare Worker) Provide scene context for the AI explanation
Up to 4 preceding subtitle lines Groq (via Cloudflare Worker) Improve contextual accuracy of the AI explanation
Show title and episode title Groq (via Cloudflare Worker) Help AI understand the show's world and characters

No data is transmitted while you are passively watching. Data is sent only in direct response to your hover interaction and is not retained on any Thesara-operated server.

2. Data we do not collect

  • Netflix account credentials — Thesara never reads or accesses your Netflix login details.
  • Viewing history or watch time — We have no access to what you watch or for how long.
  • Payment or billing information — Never accessed, never transmitted.
  • IP address or device identifiers — Not collected or logged by Thesara.
  • Personal information — No name, email, age, or demographic data of any kind.
  • Behavioural analytics — No tracking of which words you hover, how often you use the extension, or any usage patterns.
  • Crash reports or telemetry — The extension contains no telemetry code.

3. Third-party services

Thesara relies on the following third-party services to function. Each is subject to its own privacy policy, which governs how they handle the data they receive.

  • Groq, Inc. (groq.com) — Subtitle text and show context are transmitted to Groq's inference API to generate contextual word explanations using the Llama 3 language model. Groq's privacy policy: groq.com/privacy-policy.
  • Free Dictionary API (dictionaryapi.dev) — Individual words are sent to this public API to retrieve definitions, parts of speech, and synonyms. No personal data is involved. No account or registration is required.
  • Cloudflare, Inc. (cloudflare.com) — All Groq API requests are routed through a Cloudflare Worker that acts as a secure proxy to prevent the developer's API key from being embedded in the extension. Cloudflare may log request metadata in accordance with their privacy policy: cloudflare.com/privacypolicy.

4. Your optional API key

If the shared daily quota is exhausted and you voluntarily choose to add your own Groq API key, that key is governed by the following guarantees:

  • Stored locally only — saved exclusively in chrome.storage.local on your own device.
  • Never transmitted to Thesara — the developer has no access to your key at any time.
  • Only sent to Groq — used solely to authenticate requests to api.groq.com.
  • Not synced — chrome.storage.local is not synced to Google's servers or any other device.
  • Sandboxed — no website or other extension can access it; Chrome's extension isolation prevents this.
  • Revocable at any time — you can clear it from the extension's settings page at any time.

You are presented with a full disclosure screen, including links to Groq's pricing page, before you are asked to enter a key. No key is collected without your informed, explicit action.

5. Chrome permissions

Thesara requests the minimum permissions necessary to function. Each permission is used only as described below and for no other purpose.

Permission Why it's needed
storage To optionally save your personal Groq API key locally on your device.
netflix.com (host) To read subtitle text from the Netflix player when you hover a word. The extension only runs on /watch/ URLs.
api.groq.com (host) To send requests directly to Groq when a personal API key is in use.
dictionaryapi.dev (host) To fetch word definitions and synonyms.
thesara-proxy.thesara.workers.dev (host) To send requests through the Cloudflare Worker proxy using the shared API key.

6. Data retention

Thesara operates no databases and retains no user data. The Cloudflare Worker processes requests in memory and does not log or store the content of requests beyond what Cloudflare itself retains per their own policy.

The only data persisted locally is your optional Groq API key in chrome.storage.local, which remains until you manually clear it or uninstall the extension.

7. Children's privacy

Thesara is not directed at children under the age of 13 and does not knowingly collect personal data from children. If you believe a child has provided data through this extension, please contact us via the GitHub repository.

8. Changes to this policy

We may update this Privacy Policy from time to time to reflect changes in the extension's functionality or applicable law. Material changes will be indicated by an updated "Last updated" date at the top of this page.

Because Thesara is open source, all changes to this policy are publicly visible in the repository's commit history.

9. Contact

If you have questions, concerns, or requests regarding this Privacy Policy or the data practices of Thesara, please open an issue on the GitHub repository.
