Privacy Policy

Last updated: June 2026  ·  Effective immediately

Thesara is a free, open-source Chrome extension that helps you understand words and phrases while watching Netflix. This policy explains clearly and completely what data leaves your device, where it goes, and what happens to it.

Short version: Thesara sends subtitle text to AI and dictionary services only when you hover a word. Nothing is stored on any server we control. No accounts. No tracking. No ads.

1. What data is sent when you use Thesara

When you hover a word or phrase in a Netflix subtitle, the following data is sent:

  • The hovered word or phrase — sent to the Free Dictionary API for its definition, part of speech, and synonyms.
  • The current subtitle line and up to 4 preceding subtitle lines — sent to Groq's AI API to generate a contextual explanation of the word in the scene.
  • The show title and episode title (e.g. "Breaking Bad", "Ozymandias") — sent to Groq alongside subtitle text to help the AI understand the context.

No data is sent while you are simply watching. Data is only sent at the moment you hover a word.

2. What data is never collected

  • Your Netflix account details, email, or password.
  • Your viewing history, watch time, or preferences.
  • Any payment or billing information.
  • Your IP address, location, or device identifiers.
  • Any data about you as a person.

No data is ever stored on any server operated by Thesara's developer. There are no databases, no logs of your activity, and no analytics.

3. Third-party services

Thesara relies on the following third-party services to function:

  • Groq (groq.com) — Subtitle text and show context are sent to Groq's inference API (Llama 3) to generate contextual word explanations. Groq's own privacy policy governs how they handle this data: groq.com/privacy-policy.
  • Free Dictionary API (dictionaryapi.dev) — Individual words are sent for dictionary lookups. This is a public API with no accounts or personal data involved.
  • Cloudflare Workers (cloudflare.com) — API requests are routed through a Cloudflare Worker that acts as a secure proxy, keeping the developer's API key off your device. Cloudflare's privacy policy applies: cloudflare.com/privacypolicy.

4. Your optional API key

If the shared daily quota is exhausted and you choose to add your own Groq API key, that key is stored exclusively in chrome.storage.local on your own device. It is:

  • Never transmitted to Thesara's developer or any server other than Groq.
  • Never synced to the cloud or any other device.
  • Sandboxed to this extension — no website can access it.
  • Removable at any time by clearing the extension's storage.

5. Permissions

Thesara requests the following Chrome permissions and only uses them as described:

  • storage — to save your optional API key locally on your device.
  • host_permissions for netflix.com — to read subtitle text from the Netflix player page when you hover a word.
  • host_permissions for api.groq.com and dictionaryapi.dev — to make API requests for word explanations.

6. Children's privacy

Thesara does not knowingly collect or process personal data from children under the age of 13.

7. Changes to this policy

If this policy changes materially, the updated version will be published here with a new "Last updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.

Questions or concerns about this policy? Open an issue on the Thesara GitHub repository .
