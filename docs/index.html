Features

Built for how
you actually watch.

01

Instant definitions

Definition, part of speech, and synonyms on hover — without leaving the screen.

02

Scene-aware AI

"Cook" in Breaking Bad isn't about food. The AI knows the show, the characters, and the moment.

03

Idiom detection

Automatically clusters phrases like "call it a day" and explains them as a single unit.

04

Dialogue context

The last 6 subtitle lines are passed to the AI so it understands the full conversation.

Private by design

Only the subtitle you hover ever leaves your device. No history, no profiling, no accounts.

06

Zero configuration

Install and it works. No API key needed. A shared free quota is available from day one.
