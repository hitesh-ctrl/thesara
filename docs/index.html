// features

Built for serious learners

def()

Instant Definitions

Hover any word for its definition, part of speech, and synonyms — without breaking your focus or leaving the screen.

ctx.ai

Scene-Aware AI

"Cook" in Breaking Bad isn't about food. Thesara reads the show, the character, and the dialogue to explain what words actually mean.

phrase[]

Idiom Detection

Automatically groups phrases like "give it up" or "let it slide" and explains them as a unit — not word by word.

history[4]

Dialogue Context

The AI sees the last 4 lines of dialogue before the word you hovered — so it understands the conversation, not just the sentence.

privacy.ok

Private by Design

Only the subtitle you hover is ever sent anywhere. No history, no profiling, no tracking. Your viewing stays yours.

cost = 0

Free Shared Quota

14,400 AI calls per day shared across all users — provided free by the developer. Bring your own key if it runs out.

// how it works

Under the hood

thesara · runtime
// 1. SubtitleObserver watches the DOM for subtitle changes
→ observe(".player-timedtext")
 
// 2. PhraseDetector finds idioms in the current line
→ detect("give it up") ✓ idiom found
 
// 3. You hover a word — video pauses, tooltip fires
→ lookup({ word: "give it up", context: history[4] })
 
// 4. AI explains the meaning in THIS scene
→ "He's letting go of something he held onto for years…"
 
// 5. Close tooltip → video resumes
→ resume() ✓
